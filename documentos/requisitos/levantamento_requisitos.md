## 1. Identificação do Projeto

Nome:
Valor 3D

Objetivo:
Desenvolver um sistema para cálculo de custos de impressão 3D,
considerando materiais, energia elétrica, tempo de impressão,
depreciação do equipamento e mão de obra.

## 2. Problema

Atualmente muitos profissionais de impressão 3D realizam
cálculos de preço manualmente, dificultando a definição de
um valor correto para venda dos serviços.

## 3. Usuários do Sistema

- Profissionais de impressão 3D
- Pequenos negócios
- Usuários que realizam serviços de impressão

## 4. Objetivo do Sistema

Permitir calcular automaticamente o custo e sugerir um valor
de venda para peças impressas em 3D.
