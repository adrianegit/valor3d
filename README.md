# valor3d

Sistema web desenvolvido como projeto de estágio para cálculo de custos de impressão 3D.

## Objetivo

Auxiliar profissionais e empreendedores da área de impressão 3D na formação de preços, considerando:

- Custo do material
- Consumo de energia elétrica
- Mão de obra
- Depreciação da impressora
- Margem de lucro

## Tecnologias

- Python
- Django
- MySQL
- HTML
- CSS
- Bootstrap
- Git e GitHub

## Funcionalidades Previstas

- Cadastro de materiais
- Cadastro de impressoras
- Cálculo de custos
- Histórico de cálculos
- Geração de orçamento

## Status do Projeto

Em desenvolvimento
