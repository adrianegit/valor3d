## Valor3D
Gestão inteligente para impressão 3D.
Sistema web desenvolvido como projeto de estágio para auxiliar no cálculo de custos e formação de preços de peças produzidas por impressão 3D.

## Objetivo
Auxiliar profissionais e empreendedores da área de impressão 3D na definição de preços, considerando os principais custos envolvidos no processo produtivo:
Custo do material
Consumo de energia elétrica
Mão de obra
Depreciação da impressora
Margem de lucro
